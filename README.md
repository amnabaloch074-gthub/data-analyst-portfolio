# Project Title
Customer Segmentation using K-Means Clustering
## Description
Performed customer segmentation using K-Means clustering to group customers based on their purchasing behavior and identify meaningful customer segments for targeted marketing strategies.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Process
- Imported and explored the dataset
- Data cleaning and preprocessing
- Feature selection
- Applied K-Means clustering algorithm
- Determined optimal number of clusters (Elbow Method)
- Visualized the clusters using graphs

  ## key insight
- Customers were grouped into distinct clusters based on their behavior
- Identified high-value and low-value customer segments
- Revealed patterns in spending and purchasing behavior
- Helps in targeted marketing and customer strategy
